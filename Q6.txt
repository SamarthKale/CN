Wireshark

Wireshark is a network packet analyzer used to capture and monitor network traffic in real time.


-----------------------------------
Protocols in Wireshark
-----------------------------------

1. TCP (Transmission Control Protocol)
   - Reliable and connection-oriented protocol.
   - Used for web browsing and file transfer.

2. UDP (User Datagram Protocol)
   - Fast and connectionless protocol.
   - Used in streaming and gaming.

3. ARP (Address Resolution Protocol)
   - Converts IP address into MAC address.

4. ICMP (Internet Control Message Protocol)
   - Used for error reporting and ping.


-----------------------------------

tcp
